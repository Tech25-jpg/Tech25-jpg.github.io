CSS - Cascading Style Sheets

CSS is the language used to style and layout web pages. It allows you to control the color, font, spacing, and overall appearance of your website.

In this section, you'll learn about CSS syntax, selectors, properties, and how to link CSS to your HTML document.

ID selectors target a specific element with a unique identifier. They are defined using the '#' symbol followed by the ID name. For example, #box-1 will select the element with the ID of 'box-1'.

Class selectors target one or more elements that share the same class attribute. They are defined using the '.' symbol followed by the class name. For example, .box will select all elements with the class of 'box'.

Type selectors target elements based on their HTML tag name. For example, div will select all div elements.

Understanding CSS Syntax

CSS syntax consists of selectors and declarations. Selectors are used to target HTML elements, while declarations define the styles for those elements.

A CSS declaration is made up of a property and a value, separated by a colon (;). For example, color: red; sets the text color to red. Multiple declarations can be grouped together within curly braces to form a CSS rule.

    h1 {
        color: blue; /* Property: color, Value: blue */
        font-size: 24px; /* Property: font-size, Value: 24px */
        text-align: center; /* Property: text-align, Value: center */
    }

In the example above, the h1 selector targets all heading level 1 elements, color is the property, blue is the value. The declarations within the curly braces define their styles.

Writing and implementing CSS rules is a fundamental skill for creating visually appealing websites. It is also essential for responsive design as CSS allows for flexible layouts that adapt to different screen sizes (PC, Mobile, etc). Without CSS, web pages would be plain and unattractive for consumers to utilize.

Like CSS, there are rules that follow strict anatomies in order for them to be displayed correctly. Missing a single semicolon (;) or bracket can spell disaster for the rest of your CSS layout. This is why it is important to differentiate and properly structure your CSS code in case you take on a client of your own someday.

Text and Fonts

CSS provides various properties to control the appearance of text and fonts. You can set the font family, size, weight, style, and more.

For example, the font-family property allows you to specify the typeface for your text. You can use a specific font name or a generic family like serif, sans-serif, or monospace. The font-size property controls the size of the text, while font-weight can make the text bold or light.

Here's an example of how to style text using CSS:

    p {
        font-family: Arial, sans-serif; /* Sets the font family */
        font-size: 16px; /* Sets the font size */
        font-weight: bold; /* Makes the text bold */
        color: #333; /* Sets the text color */
    }

Inline CSS

Inline CSS is used to apply styles directly to an HTML element using the style attribute.

To use the method of inline CSS, simply add a style attribute to the HTML element you want to style.

    <h1 style="color: blue; font-size: 24px;">Heading</h1>

In the example above, the style attribute is used to set the color and font size of the heading. While inline CSS can be useful for quick styling, it is generally recommended to use external or internal CSS for better maintainability and separation of concerns.

Think of it as attaching a 'sticky note' to a specific item. It will only affect one specific element and ignore the others.

Inline CSS can be useful for quick styling, but it is generally recommended to use external or internal CSS for better maintainability and separation of concerns. This is because inline styles can make your HTML code cluttered and harder to read, especially if you have a lot of styles applied to different elements. It should only be used sparingly.

External CSS

External CSS is a separate file that contains all the CSS rules for a website. It is linked to the HTML document using the <link> element in the <head> section.

To create an external CSS file, simply create a new file with a .css extension (e.g., styles.css, default.css) and add your CSS rules to it. Then, link the CSS file to your HTML document using the following code:

    <link rel="stylesheet" href="styles.css">

You can connect the file to your HTML document by placing it in the same directory or specifying the correct path whether absolute or relative. By doing so, it will make your site have easier access to the color variety and remain consistent to its branding.

External CSS can be used for:

  • Styling multiple pages consistently
  • Improving maintainability and organization
  • Enhancing performance through caching (faster loading times)

Internal CSS

Internal CSS is defined within the <style> element in the <head> section of the HTML document. It allows you to write CSS rules that apply to the entire document without needing an external file.

To use internal CSS, simply add a <style> element in the <head> section and write your CSS rules inside it. For example:

    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
    </style>

Internal CSS is useful for:

  • Applying styles to a single page
  • Testing and prototyping styles quickly
  • Overriding external CSS rules for specific pages
